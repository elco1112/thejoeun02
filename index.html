커리큘럼

취업성공과 핵심인재를 만드는 압도적 커리큘럼

SUBJECT 01

웹퍼블리셔(웹디자인)

웹 퍼블리셔는 정적인 페이지를 만드는 것을 넘어 사용자의 상호작용을 처리하는 프론트엔드 개발의 영역까지 업무를 확장하고 있습니다. 이에 따른 강사들의 실무에 중요한 커리큘럼을 바탕으로 핵심인재를 양성합니다.

SUBJECT 02

UI/UX 디자인

사용자가 디지털 기기(웹, 앱 등)를 편리하게 사용하도록 시각적 요소(UI)와 전반적인 사용자 경험(UX)을 설계하고,사용자 니즈 분석, 화면 구조 설계, 프로토타이핑을 통해 비즈니스 문제를 해결하는 능력을 기릅니다

SUBJECT 03

프론트엔드

웹사이트나 앱에서 사용자가 직접 눈으로 보고 상호작용하는 화면(UI/UX)을 개발하는 기술입니다. HTML, CSS, JavaScript를 기반으로 화면 구현, 성능 최적화, API 연동 등을 담당하며, 리액트(React) 같은 프레임워크가 주로 사용되며 프론트엔드로서의 역량을 끌어올립니다.

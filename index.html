SUBJECT 01

[디지털디자인] UI/UX 웹디자인&웹퍼블리셔 양성(Figma)

- 사용자 경험(UX) 분석을 기반으로 서비스 컨텐츠의 정보구조, UI 설계, 화면설계 등 기획 및 디자인 역량을 함양

- 다양한 디지털 환경과 디바이스에 대응 가능한 컨텐츠를 반응형 웹으로 설계·디자인할 수 있는 실무 능력을 습득

- 웹표준 기반의 HTML, CSS, JavaScript 등 웹 구현 기술을 활용하여 웹디자인과 퍼블리싱 전 과정을 수행

·스토리보드 설계·제작·응용 → 심미성 구성요소 설계·제작·응용 → 사용성 구성요소 설계·제작·응용 → 매체성 구성요소 설계·제작·응용

·관찰조사 - 사용자 유형 분류 → 관찰조사 설계 → 사용자 관찰조사

·시나리오 개발 - 서비스·경험 시나리오 기획 → 제작 → 평가 보완

·시장환경 조사·분석 → 디자인트렌드 조사·분석 → 사용자 조사·분석

·비주얼 아이데이션 - 아이디어 구상·전개 → 아이디어 스케치 구상·전개 → 비주얼 방향 구상·전개

·시안 디자인 - 시안 개발 확장 → 아트워크 고도화 → 베리에이션 완료

·UX 설계 - UI 정보구조 설계 → 워크플로우 제작 → 내비게이션 설계

·UI 디자인 - 디자인 컨셉 수립 → 스타일가이드 제작 → 시안디자인 제작

·프로토타입 수집·스케치 - 기초데이터 수집 → 레퍼런스 조사·분석 → 아이디어 스케치

·프로토타입 제작·테스트 - 프로토타입 제작 → 사용성 테스트 → 테스트 수정사항 반영

·산출물 자료정리·정리·제출 → 프로젝트 결과 자료정리·보고서작성·제출 → 클라이언트 최종 보고 자료정리 / 최종 발표자료 작성 / 최종 보고

·Javascript&jQuery → 피그마 기초 및 활용 → 반응형 웹페이지 제작

SUBJECT 02

(스마트웹&콘텐츠 개발) 웹퍼블리셔(웹디자인)&프론트엔드 개발과정

다양한 종류의 디지털 환경에서 제공되는 서비스·콘텐츠를 사용자 중심의 반응형 인터페이스(UI)와 사용자 경험(UX)에 맞게 디자인하고 최적화하여
효과적으로 구현할 수 있는 전문인력 양성을 목표

·UI/UX 요구분석 - UI/UX 컨셉 아이디어 도출 → 패스트 프로토타입 제작 → 패스트 프로토타이핑 검증

·UI/UX 컨셉기획 - UI/UX 컨셉 도출 → 기능(FEATURE) 리스트 도출 → 컨텐츠 기획

·UI 아키텍처 설계 - 정보 설계 → 와이어프레임 작성 → 태스크 플로우 작성

· UI 설계 검토 → UI 구현 표준 검토 → UI 저작도구 활용 → UI 제작

·개별환경 구축 - 앱 개발 환경 선정 → 앱 개발 방법론 선정 → 앱 개발 계획 수립

·시스템 설계 - 리소스 준비 → 정보구조 설계 → 개발시스템 설계

·UX 설계 - UI 정보구조 설계 → 워크플로우 제작 → 내비게이션 설계

·UI 디자인 - 디자인 컨셉 수립 → 스타일 가이드 제작 → 시안 디자인 제작

·스마트문화앱 구현 - 프로그래밍 계획 수립 → 프로그래밍 연동 → 멀티미디어 연동 → 산출물 작성

·스마트문화앱 테스트 - 테스트 계획 수립 → 앱 테스트 수행 → 스마트 문화 앱 등록

·프로그래밍 언어활용 - 구조적 프로그래밍 언어 활용 → 스크립트 언어 활용

·응용SW 기초기술 활용 - 네트워크 기초 활용 → 미들웨어 기초 활용 → 데이터베이스 기초 활용

·SQL 활용 - 기본 SQL 작성 → 고급 SQL 작성

·서버프로그램 구현 - 개발환경 구축 → 공통 모듈 구현 → 서버 프로그램 구현 → 배치 프로그램 구현

·화면구현 - UI 설계 확인 → UI 구현

·애플리케이션 테스트 관리 - 애플리케이션 테스트 케이스 설계 → 통합 테스트 → 성능 개선

·애플리케이션 테스트 수행 - 애플리케이션 테스트 수행 → 결함 조치

·애플리케이션 배포 - 애플리케이션 배포환경 구성 → 소스 검증 → 빌드 및 배포
